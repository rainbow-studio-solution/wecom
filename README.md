# 企业微信 For Odoo

## 介绍
1. 在Odoo中封装了相关企业微信API，便于开发者在Odoo中直接调用。
2. 集成了通讯录同步、自建应用等功能。
3. 企业微信专用的 `widget`

### 使用说明

1.  注意注册 企业Id 应用id 和密钥
2.  注意设定企业微信资源存储路径用于存储企微资源的路径，需要对路径具有读写权限
3.  在获取JSAPI临时凭证可以先尝试手动获取确认一切正常在进行后续.
